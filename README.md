# react
Minimal React, Webpack, Babel Setup with Jest, Enzyme and ESLint

###### FEATURES #######
* React 16
* Webpack 4
* Babel 7
* Jest 
* Enzyme 
* ESLint

###### INSTALLATION ######
- git clone https://github.com/fenotiana/react-webpack.git
- cd react-webpack
- yarn install
- npm start
- visit http://localhost:8080/
